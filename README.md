# M1 Macbook Setup

## CLI Install

Run `install.sh` to install everything.


### Install XCode Command line Tools

```
sudo xcode-select --install
```

### Install Homebrew

```
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"
```

## Tools

* Raycast
* oh-my-zsh+p10k
* Dozer
